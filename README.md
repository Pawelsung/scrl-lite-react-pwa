# SCRL Lite React PWA

這是用 **React + Konva.js + PWA** 做的續作版原型，針對你提到的三個問題做了修正：

1. **等比縮放**  
   - 選取圖片後，用角點拖曳會維持比例。  
   - 也支援 **Ctrl/⌘ + 滑鼠滾輪** 對選取物件做等比放大縮小。

2. **主畫布看不到預覽**  
   - 改成 Konva Stage 為主畫布。
   - 畫布固定使用真實設計尺寸，視窗中只是縮放顯示，因此上方主畫布會正常渲染。

3. **版型風格更多元**  
   - 內建 4 種模板：
     - 雜誌留白
     - 左右文圖
     - 底片框
     - 大標題封面

## 啟動方式

```bash
npm install
npm run dev
```

## 功能

- 輪播張數 2–10 張
- IG 比例切換
- 多圖上傳
- 圖片加入畫布
- 文字新增
- 主畫布編輯
- 切片預覽
- 全部下載
- JSON 專案匯入 / 匯出
- 可安裝為 PWA
